## 🔧 Installation & Upgrade

### For New Users

```bash
npm install @h4shed/mcp@1.2.0

# Initialize swarm session
npm run swarm

# Verify installation
npm run build && npm run lint && npm run typecheck
```

### For Existing Users (Upgrade from v1.1.2)

```bash
npm update @h4shed/mcp

# Clear old swarm state
rm -rf .claude-flow/swarm/

# Reinitialize
npm run swarm
```

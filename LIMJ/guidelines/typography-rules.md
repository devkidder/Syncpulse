# Typography Rules

## Font Selection (Open Source Only)

All fonts must be from Google Fonts. No proprietary fonts.

### Display/Headlines (Bold, Distinctive)

Good choices:
- Space Grotesk
- Bebas Neue
- Playfair Display
- Oswald
- Archivo Black
- Anton
- DM Serif Display
- Orbitron

Bad choices (overused, AI-detectable):
- Inter
- Roboto
- Arial
- Open Sans
- Montserrat (overused on templates)

### Body/Subtext (Readable, Complementary)

Good choices:
- Source Sans Pro
- Lato
- Work Sans
- Karla
- Nunito Sans
- Raleway
- DM Sans
- Rajdhani

Bad choices:
- Helvetica (too generic)
- Times New Roman (dated)
- Comic Sans (never)

## Hierarchy Rules

For LinkedIn cover (1200×627px):

```
Headline:        48-72px bold
Subheadline:     24-32px regular
Accent/Stats:    18-24px bold
Fine Print:      14-16px regular
```

For LinkedIn article preview text:

```
Display:         Bold, distinctive, 2-3 words maximum
Supporting:      Clear, readable, provide context
```

## Pairing Examples

**Tech/Modern**:
- Display: Space Grotesk Bold
- Body: DM Mono

**Editorial/Sophisticated**:
- Display: Playfair Display Bold
- Body: Source Sans Pro Regular

**Gaming/Bold**:
- Display: Bebas Neue
- Body: Raleway

**Legal/Professional**:
- Display: Oswald
- Body: Karla
